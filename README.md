# Newton School Platform Clone 🚀

A frontend clone of the Newton School student dashboard, built to replicate the UI/UX and core features of the learning management system. This project features a responsive design, interactive widgets, and dynamic mock data for a complete student platform experience.

## ✨ Features

- **Interactive Dashboard**: A comprehensive main feed displaying upcoming deadlines, latest released tasks, and course progress.
- **Navigation Sidebar**: Quick access to Arena, Calendar, Leaderboard, Scorecard, and Timeline.
- **Task & Deadline Tracking**: Visual indicators for assignments (In Class, Post Class) with deadlines and XP multipliers.
- **Dynamic Widgets**: "Question of the Day" and Calendar schedule widgets on the right sidebar.
- **Mock Data Integration**: Pre-populated with realistic courses (DSA, WAP, FOAI, Maths) to demonstrate UI states.
- **Responsive Layout**: Clean, modern CSS styling for optimal viewing.

## 🛠️ Technologies Used

- **React 19**: Modern UI component architecture.
- **Vite**: Lightning-fast build tool and development server.
- **Vanilla CSS**: Custom, tailored styling across the application without relying on heavy frameworks.
- **Lucide React**: Beautiful and consistent SVG icons.
- **ESLint**: Code linting and formatting.

## 🚀 Getting Started

To get a local copy up and running, follow these simple steps:

### Prerequisites

- Node.js (v18 or higher recommended)
- npm or yarn

### Installation

1. Clone the repository
   ```sh
   git clone <repository-url>
   ```
2. Navigate to the project directory
   ```sh
   cd NewtonSchool_Clone
   ```
3. Install dependencies
   ```sh
   npm install
   ```
4. Start the development server
   ```sh
   npm run dev
   ```
5. Open your browser and visit `http://localhost:5173`
